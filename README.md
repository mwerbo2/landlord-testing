# Landlord

Landlord is a review platform for rental units.

## Installation

Using Docker: 

```bash 
docker run ...
```

From [Github](https://github.com/mwerbo2/landlord.git):

```bash
npm install client
```
```bash
npm install api
```
```bash
npm install dev
```
## Technologies
Frontend:
React.JS, Axios, SemanticUI

Backend: Node.js, Express, MySQL
